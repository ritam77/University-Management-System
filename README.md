# University-Management-System

Front-End : HTML,CSS,JS
Back-End  : JSP(JAVA Enterprise Edition)
Database : Mysql
Server : Apache Tomcat 10.0(Localhost)
IDE : Eclipse 

Key Feature : Multiple login access(admin,teach,student)
